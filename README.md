# DESKRIPSI PROJECT

Tetris, sebuah permainan puzzle klasik yang menantang pemain untuk menyusun balok-balok berbentuk geometris (tetromino) yang jatuh dari atas layar ke dalam area permainan berbentuk kotak. Tujuan utama permainan ini adalah menyusun balok agar membentuk barisan horizontal penuh, sehingga barisan tersebut akan menghilang dan pemain memperoleh skor. Di sisi kiri ditampilkan area utama permainan, sementara di bagian atas terdapat informasi Score, Lines, dan Level yang menunjukkan perkembangan pemain. Di sisi kanan, terdapat panel Next Piece yang memperlihatkan balok berikutnya agar pemain dapat menyusun strategi. Kontrol permainan menggunakan tombol arah untuk menggerakkan balok, tombol rotasi untuk memutar balok, serta fitur soft drop dan hard drop untuk mempercepat jatuhnya balok. Dengan tampilan modern bernuansa gelap dan kontrol yang responsif, game ini melatih konsentrasi, kecepatan berpikir, dan kemampuan strategi pemain.

# YANG DIBUTUHKAN OLEH PROJECT INI

Proyek game Tetris ini membutuhkan teknologi web berupa HTML5 sebagai struktur halaman, CSS3 untuk desain antarmuka yang menarik dan responsif, serta JavaScript untuk mengatur seluruh logika permainan. Penggunaan Canvas API diperlukan untuk menggambar papan permainan, tetromino, dan animasi secara real-time. Selain itu, proyek ini memerlukan perangkat keras berupa komputer atau laptop dengan keyboard dan browser modern sebagai media menjalankan game. Dari sisi sistem, dibutuhkan komponen permainan seperti papan grid, berbagai bentuk tetromino, sistem skor, level, garis yang dihapus, fitur next piece, serta mekanisme kontrol, rotasi, deteksi tabrakan, dan game over. Keseluruhan kebutuhan tersebut saling terintegrasi untuk menghasilkan permainan Tetris yang berjalan dengan baik, interaktif, dan nyaman dimainkan oleh pengguna.

# HOW TO USE

Untuk menggunakan game Tetris ini, pengguna cukup membuka halaman game melalui web browser pada komputer atau perangkat mobile. Setelah halaman terbuka, permainan akan dimulai ketika pemain menekan tombol kontrol pertama kali. Balok (tetromino) akan jatuh dari bagian atas papan permainan dan dapat dikendalikan menggunakan tombol panah kiri dan kanan untuk menggeser balok, tombol panah atas (↑) untuk memutar balok, dan tombol panah bawah (↓) untuk mempercepat jatuhnya balok (soft drop). Pemain juga dapat menekan spasi (Space) untuk menjatuhkan balok secara langsung ke bawah (hard drop) dan tombol P untuk menjeda atau melanjutkan permainan. Tujuan permainan adalah menyusun balok agar membentuk satu baris horizontal penuh sehingga baris tersebut akan menghilang dan skor bertambah. Permainan akan berakhir (game over) jika balok tidak lagi memiliki ruang untuk muncul di bagian atas papan permainan.
